### Proxy Server

Proxies requests with the `Access-Control-Allow-Origin : *` header


### Usage
```sh
 GET http://proxy-server.herokuapp.com/[URL]
```

e.g.  
```sh 
GET http://proxy-server.herokuapp.com/http://safaricom.com
```
